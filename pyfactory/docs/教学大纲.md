## 一、Python语言基本语法元素

### 1.1 程序的基本结构 (关卡: basics_01)
**考纲要求：** 了解程序的三种基本结构

**游戏对应：**
- 源头 → 输入
- 加工机器 → 处理
- 输出口 → 输出

**Python示例：**
```python
# 顺序结构
shape = input()    # 输入
processed = process(shape)  # 处理
output(processed)  # 输出
```

### 1.2 变量与赋值 (关卡: variables_01, variables_02)
**考纲要求：** 掌握变量的命名规则和赋值操作

**游戏对应：**
- 储物箱 = 变量
- 放入物品 = 赋值
- 取出使用 = 引用变量

**Python示例：**
```python
# 变量赋值
box = Circle("white")
shape1 = box
shape2 = box
```

## 三、程序的控制结构

### 3.1 条件分支 (关卡: conditionals_01, conditionals_02)
**考纲要求：** if-elif-else语句

**游戏对应：**
- 分拣器机器
- 根据图形属性（颜色、形状）分流

**Python示例：**
```python
if shape.color == "red":
    output_a(shape)
elif shape.color == "blue":
    output_b(shape)
else:
    output_c(shape)
```

### 3.2 循环结构 (关卡: loops_01, loops_02)
**考纲要求：** for循环、while循环、break/continue

**游戏对应：**
- 循环器机器
- 复制和批量加工

**Python示例：**
```python
# for循环
for i in range(5):
    shape = Circle("red")
    output(shape)

# 带变换的循环
for i in range(4):
    shape = Circle("red")
    shape.rotate(i * 90)
    output(shape)
```

## 四、函数和代码复用

### 4.1 函数定义 (关卡: functions_01)
**考纲要求：** def语句、函数的调用

**游戏对应：**
- 函数机器 = 可复用的加工蓝图
- 内部机器组合 = 函数体

**Python示例：**
```python
def make_red_circle():
    shape = Circle("white")
    shape.paint("red")
    return shape

result = make_red_circle()
```

### 4.2 函数参数 (关卡: functions_02)
**考纲要求：** 位置参数、关键字参数、默认值参数

**游戏对应：**
- 函数机器的配置面板
- 参数化的加工设置

**Python示例：**
```python
def paint_circle(color):
    shape = Circle("white")
    shape.paint(color)
    return shape

red = paint_circle("red")
blue = paint_circle("blue")
```

## 六、推导式

### 6.1 列表推导式 (关卡: advanced_01, advanced_02)
**考纲要求：** 列表推导式的语法和应用

**游戏对应：**
- 推导机
- 批量加工流水线

**Python示例：**
```python
# 基本推导式
circles = [Circle("white") for _ in range(5)]

# 带变换的推导式
red_circles = [c.paint("red") for c in circles]

# 带条件的推导式
red_only = [s for s in shapes if s.color == "red"]
```

## 八、学习路径建议

### 阶段一：入门（1-2小时）
1. 完成基础入门关卡
2. 熟悉游戏操作
3. 理解输入-处理-输出的基本结构

### 阶段二：基础（2-3小时）
1. 学习变量和赋值
2. 掌握条件分支
3. 理解循环结构

### 阶段三：进阶（3-4小时）
1. 学习函数定义和调用
2. 掌握数据结构（列表、字典）
3. 学习解包操作

### 阶段四：高级（2-3小时）
1. 掌握列表推导式
2. 综合应用各种知识点
3. 挑战沙盒模式创意建造
